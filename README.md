# Next.js & TailwindCSS starter template

A starter for Next.js (v13) and Tailwind CSS (v3).

Keywords:

- TypeScript
- React
- Next.js
- TailwindCSS
- ESLint
- Prettier

### `Before launch`

```
yarn install
```

### `develop`

```
yarn develop
```

### `build`

```
yarn build
```

### `start`

```
yarn start
```
